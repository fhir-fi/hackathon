# HL7 Finland: ATK-hackathon

This repo contains the source code for the website for the hackathon event organized in connection
with the [Sosiaali- ja terveydenhuollon ATK-päivät event](https://koulutus.fcg.fi/sosiaali-ja-terveydenhuollon-ATK-paivat).

## Development

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

First, install everything:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.
